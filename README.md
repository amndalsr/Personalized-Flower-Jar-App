# Personalized Flower Jar App

An interactive flower arrangement builder. Choose from 12 flower types and 8 different vases to compose your personalized bouquet.

<img width="1057" height="708" alt="image" src="https://github.com/user-attachments/assets/17c98c09-46b0-4055-999a-a9e32ffe8a14" />

## Features

- 12 hand-drawn flowers with sketch SVG effect (feTurbulence filter)
- 8 vases in varied styles (terracotta, crystal, striped, amber...)
- Drag flowers to reposition them
- Click a flower to remove it
- Clear arrangement button

## Getting started

```bash
npm install
npm run dev
```

## Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS v4
- Radix UI (Dialog, Tabs)
- SVG with feTurbulence filters for sketch effect
